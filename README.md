## Setup

1. Install dependencies:
```bash
npm install
```

2. Build TypeScript:
```bash
npm run build
```

3. Open `index.html` in your browser

## How to Play

1. Select a difficulty level
2. Try to hover over or click the button
3. The button will run away each time you get close
4. Catch it if you can!

## Difficulty Modes

- **Easy**: Small movements within a limited range
- **Medium**: Moderate movements with more distance
- **Hard**: Teleports anywhere on the screen
